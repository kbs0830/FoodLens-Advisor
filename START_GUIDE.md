🚨 常見問題
════════════════════════════════════════════════════════════

Q1: 雙擊 .bat 沒反應?
A: 右鍵 → 編輯 → 複製所有内容 → 在 cmd 中貼上執行

Q2: "Python not found" 錯誤?
A: 確保 Python 已安裝並添加到 PATH
   驗證: 在 cmd 中輸入 python --version

Q3: 埠 8000 或 8080 已在使用?
A: 找到佔用進程並終止
   netstat -ano | findstr "8000\|8080"
   taskkill /PID <PID> /F

Q4: 前端無法連接後端?
A: 確保兩個服務都已啟動
   檢查防火牆設置
   確認後端 API 地址正確

Q5: 如何停止服務?
A: 在對應終端窗口按 Ctrl+C
   兩次 Ctrl+C 確保完全停止

# Web Client

簡潔的 HTML + CSS + JavaScript 前端，用來連接 FoodLens BFF。

## 快速開始

1. 確保 BFF 已啟動：
```bash
python app/main.py
```

2. 在瀏覽器開啟 `index.html`（或用本地伺服器）：
```bash
python -m http.server 8000
# 然後打開 http://127.0.0.1:8000
```

3. 點擊「測試 BFF 連線」按鈕

## 功能

- ✅ 連線測試
- ✅ 食物分析
- ✅ 營養統計展示
- ✅ 標準檢查
- ✅ 建議顯示

## 技術

- HTML5
- CSS3（漸層、動畫）
- Vanilla JavaScript（Fetch API）
